# ERI Scan Face – Face Recognition Security

## Overview
A biometric system for registering and verifying students using facial recognition for secure exam access.

## Features
- Face registration
- Real-time verification
- Anti-cheating authentication
- Student database integration

## Tech Stack
- Java (Android)
- ML Kit / Face Recognition API
- SQLite / PHP backend

## Use Case
- Exam security
- Identity verification
- Access control systems
